# hsleiden-ipsenh-api
Api for IoT platform created during IPSENH - groep 2

## Getting started
```sh
# Start via makefile
$ make start 
# Or start via docker-compose
$ docker-compose up -d
```

**Testing**
```sh
$ make suite
```
